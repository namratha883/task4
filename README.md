# 📝 Smart To-Do Manager

A modern and interactive To-Do Web Application built using **HTML, CSS, and JavaScript**. It helps users manage daily tasks efficiently with features like task creation, editing, deletion, priority levels, task pinning, dark mode, progress tracking, search functionality, and local storage support.

## 🚀 Features

* Add, Edit, and Delete Tasks
* Mark Tasks as Completed
* Priority Levels (High, Medium, Low)
* Pin Important Tasks
* Search Tasks
* Progress Bar & Task Statistics
* Dark Mode Toggle
* Due Date & Time Support
* Local Storage for Data Persistence
* Responsive and User-Friendly Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 🎯 Purpose

This project demonstrates DOM manipulation, event handling, responsive design, and local storage implementation while providing a practical task management solution.
